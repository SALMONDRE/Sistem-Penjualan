# Sistem Penjualan
Aplikasi desktop Sistem Penjualan yang dibuat menggunakan VB.Net dan SQL Server 2022

## Fitur
- Login
- Data Barang
- Data User
- Data Kategori
- Transaksi Penjualan
- Laporan
- Keluar

## Tools 
- Visual Studio 2022
- SQL Server 2022

## Cara menjalankan
1. Inport database ke SQL server 2022
2. Buka file .sln menggunakan visual studio 2022
3. Jalankan aplikasi
